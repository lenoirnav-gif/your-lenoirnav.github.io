Web Application Programming and Hacking (WAPH)

Instructor: Dr. Phu Phung

Course Overview

This academic course delves deeply into the architectures, programming models, vulnerabilities, defensive patterns, and hands-on tools central to creating secure enterprise web systems. Key focuses include hands-on capture-the-flag validation, scripting utilities, engineering standards, and vulnerability mitigation.

Assigned Hands-on Assignments

  • Lab 1: Virtual Machine environments, deployment architectures, and network protocols setup.
  • Lab 2: Front-end interaction architectures utilizing jQuery, local asynchronous data parsing, and State retention tracking models.
  • Individual Project 1: Secure hosting, environment scaling, integration configurations via GitHub Pages, and component validation.
